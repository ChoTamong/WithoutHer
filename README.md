## ❓ 프로젝트 개요
한 남자의 절박한 선택과 그로 인한 비극적 여정을 다룬 게임 
- 게임 이름: Without Her
- 장르: 텍스트 기반 RPG
- 컨셉: 선과 악의 경계에서 고뇌하는 한 남자의 감옥 탈출기
- 특징: 선택에 따라 달라지는 결말, 모든 엔딩이 비극으로 끝나는 독특한 구조
